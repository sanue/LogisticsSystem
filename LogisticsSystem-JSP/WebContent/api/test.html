物流管理システム REST API 测试

商品管理 API

GET /api/products
获取商品列表（支持分页和搜索）
参数： - page: 页码（默认0） - size: 每页大小（默认10） - sortBy: 排序字段（默认CREATED_AT） - sortDir: 排序方向（默认DESC） - keyword: 搜索关键词 示例：/api/products?page=0&size=5&keyword=商品
GET /api/products/{id}
根据ID获取商品详情
POST /api/products
创建新商品
请求体： { "productId": "P001", "productName": "测试商品", "specification": "规格说明", "unit": "个", "safetyStock": 100 }
PUT /api/products/{id}
更新商品信息
DELETE /api/products/{id}
删除商品
